# i3wm-kali-desktop

fast install i3wm-kali-desktop


## Install
```bash
git clone https://github.com/noob-Engle/i3wm-kali.git
./install
```
after the script is done reboot and select i3 (top right corner) on the login screen

open alacritty `ctrl+return`
